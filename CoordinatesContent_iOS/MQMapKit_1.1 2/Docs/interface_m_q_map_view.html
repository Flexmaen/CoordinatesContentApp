Detailed Description

The MapView: Loads map tiles and provides zooming, panning, overlays, and point annotation handling.

Property Documentation

- (CLLocationCoordinate2D) centerCoordinate
readwriteatomic

The center lat/lon of the displayed map. Setting this property changes the viewed portion of the map immediately. To change the center with animation, use setCenterCoordinate:animated instead.

- (MQMapType) mapType
readwritenonatomicassign

Returns the type of map tiles being displayed

- (MQCoordinateRegion) region
readwritenonatomicassign

The MQCoordinateRegion currently displayed in the map view. Setting this property causes the viewed region to change immediately. To animate the change, use setRegion:animated instead;

- (BOOL) scrollEnabled
readwritenonatomicassign

Returns true if panning (vai touch events) is enabled.

Warning

- (NSArray*) selectedAnnotations

Warning

- (BOOL) showsHeading
readwritenonatomicassign

Determines whether the map is allowed to display heading. Setting this property to YES causes the map to begin tracking heading and rotating accordingly

- (BOOL) showsUserLocation
readwritenonatomicassign

Determines whether the map is allowed to display user location. Setting this property to YES causes the map to begin tracking the user's location using CLLocationManager

- (MQMapRect) visibleMapRect
readwritenonatomicassign

The portion of the map that is visible to the user. This is represented as an MQMapRect roughly conformed to screen dimensions

Warning
Setting this property will result in an approximation of the passed-in value, accounting for best-fit scaling
- (BOOL) zoomEnabled
readwritenonatomicassign

Returns true if map zooming (via multitouch events) is enabled.

Warning
The zoom level can be programatically set wither this property is true or false
