Detailed Description

This class is poorly named... sorry. Its basically a wrapper around results and current state of searches managed by the MQSearchManager. Its the glue allowing other pieces to be decoupled easily.

NOTE: This is only intended to be used exclusivly in the MQSearchManager... If it leakes out somewhere we need to re-evaluate how this is put together
