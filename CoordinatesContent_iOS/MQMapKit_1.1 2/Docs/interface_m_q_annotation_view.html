The primary initializer. Create MQAnnotations with this method only, or be sure to call [super] from within custom subclasses.

Parameters
annotationThe annotation that is associated with the view. Cannot be nil, but may be changed. If changed, remove and re-add the annotation to the map.
reuseIdentifierThe string used to cache unused views. Used in the dequeue method of the MQMapView.

- (void) setSelected: (BOOL)  selected
animated: (BOOL)  animated 

Marks the annotation as selected within its internal state.

Warning
The callout will not display simply by calling this method. Call the method MQMapView selectAnnotation:animated instead.
Parameters
selected- Mark the annotation as being in the selected state. The meaning of this is determined by the annotation view.
animated- If changes to the selected state imply visual changes, it is up to the annotation view implementation to render those as animations or not.
